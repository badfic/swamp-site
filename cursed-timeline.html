The Story of How We Became Cursed

THE FOLLOWING IS THE REAL LIFE ACCOUNT OF HOW RARE PAIR AFFAIR WENT FROM BEING AN INNOCENT RARE PAIR SERVER TO THE CURSED SERVER IT IS TODAY

 
 
 
 
 
Wednesday August 5th, 2020

Day 0

Bored at work Wren decides to play around on Discord in a different server. Admin won't give her rights to invite people to the server or change anything. She decides to create her own server instead.

Thursday August 6th, 2020

Dawn of the 1st Day

Wren decides on a server concept. (Because of one Alli the Whoa Whoa titty) She decides to create a rare pair server called Rare Pair Affair where people can come to discuss their favourite rare pairs. Wren opens the server and invites Alli and Lisa to help her set up. Alli and Lisa join the fray.

 
 
 
 
 
 
 
 
 
 
Day 1 continued

Later that same day...

It is worth noting here, that Wren mentioned this originally to Nelly as a funny secret server she was working on. Nelly did nothing to stop her.

Day 1 Continues Again

Set up and invitations

After setting up roles through Role Machine and Shouty boy, Wren invites Megan as the official beta tester to see if the roles work. And also because she set up a hunger games sim channel she wants Megan to run. Megan joins the fray.

 
 
 
 
 
 
 
 
 
 
More on the first day

Day 1 Keeps on

Seven hours later, the server is set up with the idea that it will be a rare pair server to discuss Zava, Rosy (Tony Stark/Rose Tyler), Hellstar (tho it's not so rare anymore), and other pairings, ESPECIALLY Polyam ones.

The never-ending first day

The day keeps coming

Wren posts first promotional links on Twitter and Tumblr. Nelly, Kath, Amanda, Paige, and Punk join the fray.

 
 
 
 
 
 
 
 
Then FRECKLES

The conversations started tame

There is hours of tame comic book, rare pair, and meme chat. Then Freckles joins the fray and immediately as a joke requests a bastard ranking. This is the beginning of the end.
